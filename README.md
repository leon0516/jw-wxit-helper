# jw-wxit-helper
锡职教务助手。学生一键评教、教师一键录入成绩(总评成绩)、教师一键互评、教师一键评学

**支持的功能**
-------
1. 学生一键评教
2. 教师一键录入成绩(总评成绩)
3. 教师一键互评
4. 教师一键评学
